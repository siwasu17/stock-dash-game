## Stock Dash Game

### 1. プロジェクト概要
株取引をモチーフとした、リアルタイム・ダッシュボード形式のゲームアプリケーション。
5つの銘柄の価格変動を監視し、タイムラグのある注文システムを攻略して制限時間内に総資産を増やす。

### 2. 技術スタック
- Framework: React + Vite (TypeScript)
- Styling: Tailwind CSS + shadcn/ui
- State Management: Zustand
- Chart: Chart.js + chartjs-plugin-streaming
- Deployment: GitHub Pages (DBなし、LocalStorage活用)

### 3. ゲームシステム仕様
#### 3.1 銘柄設定
- 全5銘柄。それぞれ「リスク値（ボラティリティ）」が異なる。
- 銘柄間の相関関係: 
  - 銘柄Aが上がると銘柄Bも追従する。
  - 銘柄Cと銘柄Dは逆相関（一方が上がれば一方が下がる）。
  - 銘柄Eは市場全体の先行指標となる。

#### 3.2 取引ルール
- タイムラグ注文: 購入/売却ボタンを押してから、実際に約定（実行）されるまで5秒のラグがある。
- 価格更新: chartjs-plugin-streaming を用い、1秒ごとにリアルタイム更新。
- 勝敗条件: 3分間の制限時間終了時の「現金 + 株式時価評価額」の合計。

#### 4. 画面構成 (UI/UX)
- 上部：ヘッダー（残り時間、総資産、現在の損益）
- 中央：5つのチャートをグリッド配置（2x3 または 1+4構成）
- 右側または下部：注文パネル（銘柄選択、売買ボタン、待機中の注文リスト）

##### 画面遷移
1. Start画面 -> 2. 銘柄紹介画面 -> 3. メインゲーム画面 -> 4. リザルト画面


#### 5. 実装フェーズごとの指示内容
##### Phase 1: プロジェクト基盤とState管理
- useGameStore (Zustand) の作成。
- 状態管理項目：balance, holdings (保有株), stocks (現在の株価履歴), orders (待機中の注文), timer.
- 注文から5秒後に holdings を更新するロジックの実装。

##### Phase 2: チャートコンポーネント
- chartjs-plugin-streaming を使用した RealtimeChart コンポーネントの作成。
- 5つの銘柄を個別の小窓（Card）で表示。

##### Phase 3: ゲームロジック
- useEffect を使った1秒ごとの株価計算エンジン。
- 相関アルゴリズムの実装（Aの変動値をBの計算式に組み込む等）。

##### Phase 4: UIブラッシュアップ
- shadcn/ui を使用したスタイリッシュなダークモードUI。
- 売買ボタン押下時のフィードバック、リザルト画面のスコア表示。
